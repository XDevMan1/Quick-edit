Quick Edit Bookmarklet

This is a bookmarklet that make the current website editable. You can change the content around and when you hit `ESC` you go back to normal mode. Notice that this is not permanent, reloading the page will undo your changes. This should only make it easier to edit a page before creating a screenshot or similar tasks.

Drag the following link to your bookmarks bar to get the functionality.

Quick Edit
